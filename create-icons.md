# 应用图标创建指南

## 图标要求
为了完善Android应用，需要创建以下图标：

### 1. 应用启动图标
- **尺寸**: 512x512 像素
- **格式**: PNG
- **用途**: 应用商店和设备主屏幕
- **建议**: 使用跑步相关的简洁设计，包含"RunFlow"文字或跑步图标

### 2. 通知图标
- **尺寸**: 24x24 像素
- **格式**: PNG (透明背景)
- **颜色**: 白色/单色
- **用途**: 状态栏通知

### 3. 启动画面
- **尺寸**: 1080x1920 像素 (竖屏)
- **格式**: PNG
- **用途**: 应用启动时的过渡画面

## 图标生成命令
安装Capacitor资源生成工具后，可以使用以下命令：

```bash
# 安装资源生成工具
npm install @capacitor/assets --save-dev

# 生成所有平台图标和启动画面
npx cap assets generate
```

## 设计建议
1. **主色调**: 使用蓝色或绿色系，符合运动健康主题
2. **图形元素**: 跑步人物、跑道、心率图等
3. **文字**: "RunFlow" 或 "RF" 简写
4. **风格**: 现代、简洁、动感

## 文件放置位置
图标文件应放置在：
```
resources/
├── icon.png
├── icon/
│   ├── android/
│   └── ios/
└── splash.png
```

创建图标后，运行 `npx cap sync` 将资源同步到Android项目。
